# 04. Interactive Logic and UI

This folder contains projects focused on User Interface (UI) through the terminal and visual data representation.

### 🎮 Included Modules:

1. **Geometric Shape Generator** (`Geometric-Shape-Generator.c`)
   * Renders rectangles and patterns using ASCII symbols.
2. **Parabola Visualizer** (`Parabola-Visualizer.c`)
   * Visualizes quadratic equations by plotting stars in a text file.
3. **Interactive String Analyzer** (`Interactive-String-Analyzer.c`)
   * A full menu-driven application for string manipulation and encoding.
