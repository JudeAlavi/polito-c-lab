# 03. Data Structures and Processing

This section focuses on data manipulation, custom encryption logic, and advanced file I/O operations.

### 🛠 Tools Included:

1. **Encryption System** (`Encryption-System.c`)
   * Implements mirrored character encoding for basic file security.
2. **Base Conversion Tool** (`Base-Conversion-Tool.c`)
   * Converts numerical values between different bases (2-9).
3. **Text Formatter** (`Text-Formatter-Justified.c`)
   * A utility to reformat text files into a specified line-length (Justification logic).
